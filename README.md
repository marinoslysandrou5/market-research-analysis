# Market Research Data Analysis

## 📊 Project Overview
This project analyzes survey data to identify customer satisfaction trends and behavioral insights.

## 🛠 Tools Used
- Python (Pandas, Matplotlib)
- Data Analysis techniques (EDA)
- CSV data handling

## 📈 Key Analysis
- Calculated average customer satisfaction
- Analyzed satisfaction by demographic groups
- Evaluated recommendation rates

## 🔍 Key Findings
- Higher satisfaction observed among high product usage users
- Female customers show slightly higher satisfaction on average
- Recommendation rate strongly correlates with satisfaction

## 📁 Project Structure
- `data.csv` → dataset
- `analysis.py` → analysis script

## 🚀 Conclusion
This analysis demonstrates how survey data can be transformed into actionable insights to support business decision-making.
